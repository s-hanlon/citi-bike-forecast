# Citi Bike Demand Forecasting

A time-series forecasting project that predicts hourly Citi Bike pickups for high-traffic stations.

## Initial MVP

- Aggregate historical trips into hourly station-level demand
- Identify the 25 busiest stations
- Forecast hourly pickups for the next 24 hours
- Compare forecasts against simple seasonal baselines
- Visualize forecasts and model performance
